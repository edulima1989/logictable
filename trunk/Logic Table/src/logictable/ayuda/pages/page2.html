Características:

* Logic Table permite el desarrallo de ejercicios del algebra proposicional por medio de tablas de verdad

* Logic Table permite crear nuevos documentos (en nuevas pestañas) al mismo tiempo con el fin de desarrollar varios ejercicios y tener mejor comodidad de utilizarlos.

* Logic Table permite guardar sus documentos o jercicios editados, para utilizaciones posteriores.

*  Logic Table brinda para su mejor comodidad la opciones de: copiar, cortar, pegar; con el fin de que la transferencia de datos desde un lugar a otro sea rápido

*  Logic Table presenta una ayuda con el fin de guiar al Usuario en la manipulación del Programa.
